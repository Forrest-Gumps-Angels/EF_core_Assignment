***README***

Når programmet køres, tillades det at man henholdsvis kan browse dataen i databasen 
eller man kan oprette nye instanser og ligge i databasen. Altså fås menuen

"Press B for browsing database and A to add to database:"

I tilfælde af browse, kan man finde information på 
	- Lærere afhængigt af auid
	- Kurser afhængigt af navn
	- Studerende afhængigt af auid
	- Statistik over help request for kurserne

Hertil kan man se data afhængigt af bruger-input, ift. hvilken lære, studerende eller hvilket
kursus man vil danne overblik over.

I tilfælde af tilføjelse, kan man tilføje data på
	- Kurser
	- Studerende
	- Lærere
	- Opgaver
	- Øvelser
	- Help Requests

Hertil kan man tilføje data, hvor man via bruger-input kan vælge mellem tilhørende og
afhængige attributer til den tilhørende entitet.
